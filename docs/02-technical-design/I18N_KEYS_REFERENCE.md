## 🎯 使用示例

### 在组件中使用翻译

```rust
use crate::i18n::use_translation;

#[component]
fn MyPage() -> Element {
    let t = use_translation();
    
    rsx! {
        // 页面标题
        h1 { {t("page.swap.title")} }
        
        // 按钮
        button { {t("common.confirm")} }
        
        // 表单标签
        label { {t("form.email")} }
        
        // 错误提示
        p { {t("error.required_field")} }
    }
}
```
